You can use these *shortcut keys* if it's enabled in Options.

Web browse mode:

 * Ctrl + m : open sidebar, and focus on search input box.
 * Alt + z : close all project tabs.
 * Ctrl + , : goto first tab.
 * Ctrl + . : goto last tab.
 * Alt + , : goto previous tab.
 * Alt + . : goto next tab.
 * Alt + x : close all tabs.
 * Alt + n : toggle sidebar.
 * q : close current tab.
 * w : goto previous tab.
 * e : goto next tab.
 * j : scroll down by 2 lines.
 * k : scroll up by 2 lines.
 * h : scroll left by 2 columns.
 * l : scroll right by 2 columns.
 * 0 : scroll left to start.
 * $ : scroll right to end.
 * g : scroll up to top.
 * G : scroll down to bottom.
 * v : scroll down by half screen.
 * f : scroll up by half screen.
 * x : scroll down by one page.
 * s : scroll up by one page.
 * t : open a new tab, and focus on URL location bar.
 * . : goto next tab.
 * , : goto previous tab.
 * r : reload current tab.
 * d : open sidebar, and delete current item in tree.
 * a : open sidebar, and open all item in tree folder.
 * z : close all project tabs.
 * p : open project management window.


Text edit mode:

 * F5 : input date string.
 * F6 : save text.
 * Ctrl + Y : launch gvim editor.
