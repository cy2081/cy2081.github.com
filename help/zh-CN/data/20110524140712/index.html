文字编辑模式:

在 2.8 版更改: 去掉了 Ctrl - y 和 Ctrl - e ，用按钮代替。

  • Ctrl - y : 使用 Gvim 编辑器编辑文字。
  • Ctrl - e : 使用 Emacs 编辑器编辑文字。
  • F5 : 插入日期字符。
  • F6 : 保存当前的文本。
