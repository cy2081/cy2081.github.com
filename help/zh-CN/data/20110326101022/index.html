安装方法¶

  1. 安装 Firefox 浏览器 .
  2. 安装 穿越资源管理器 扩展，并重启Firefox。

如果要编辑文档并生成网页文件，则还需要：

  1. 安装 Python 语言 。大多数 Linux 之类的系统已经安装了Python，可能不需要再安装了。

  2. 安装 Sphinx 模块 ，完成后，可以找到类似如下的Sphinx命令:

    /usr/bin/sphinx-build

    Windows:

    C:\Python27\Scripts\sphinx-build.exe
