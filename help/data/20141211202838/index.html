Edit documents¶

  1. Open Firefox browser, and select the menu Baow ‣ Show in Sidebar

  2. Click the menu New Document from the sidebar, and input text, edit in new opened tab editor, and click Save button.

  3. Click button from the sidebar.

  4. Select your sphinx build command path, and output path, click Ok .

  5. OK, you got full features web documents.

Gvim editor¶

You can configure Gvim editor path in Options windows.

Shortcuts¶

Text edit mode:

F5 input date string.
F6 save text.
