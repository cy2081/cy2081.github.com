1. Quick install and start¶

Quick install step :

  1. Install Firefox web browser .
  2. Install baow add-on.

If you want to generate web pages with Sphinx tools:

  1. Install Python Programming Language .
  2. Install Python Package: Sphinx .

Quick start :

  1. Open Firefox browser, and select menu Baow ‣ Show in Sidebar

  2. Click menu New Document from the sidebar, and input title dialog, then some text in new opened tab editor, and click Save button.

  3. Click button from the sidebar.

  4. Select your sphinx build command path, and output path, click Ok . This path in Windows system like this:

    C:\Python27\Scripts\sphinx-build.exe

    In Linux or BSD system like this

    /usr/bin/sphinx-build

  5. OK, you get full features web documents. If you want to continue get more goods of Baow, you should create a project by click management button , and start your real documents writing.
