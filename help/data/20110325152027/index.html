Install¶

Windows¶

  1. Install Firefox web browser .
  2. Install baow and restart Firefox.

If you want to generate web pages with Sphinx tools:

  1. Install Python Programming Language .

  2. Install Python Package: Sphinx , or install it with:

    easy_install -U Sphinx

    You can get easy_install from Python Package Index : setuptools

    Finally, you should find sphinx command like this:

    C:\Python27\Scripts\sphinx-build.exe

Linux/BSD/Unix¶

  1. Install Firefox web browser .
  2. Install baow and restart Firefox.

If you want to generate web pages with Sphinx tools:

  1. Install Python Programming Language , maybe your system have installed with Python language.

  2. Install Python Package: Sphinx , or install it with:

    easy_install -U Sphinx

    You can get easy_install from Python Package Index : setuptools

    Finally, you should find Sphinx command like this:

    /usr/bin/sphinx-build
