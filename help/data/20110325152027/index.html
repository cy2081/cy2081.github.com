Install¶

  1. Install Firefox web browser .
  2. Install Baow Resource Manager and restart Firefox.

If you want to generate web pages with Sphinx tools:

  1. Install Python Programming Language
  2. Install Python Package: Sphinx

Finally, you should find Sphinx command like this:

/usr/bin/sphinx-build

Windows:

C:\Python27\Scripts\sphinx-build.exe
