下载安装¶

下载

当前版本是： 3.6

公布时间：2014-12-08

下载 安装文件 3.6

安装方法：

首先解压缩，把plugin目录下的文件复制到 Vim 的 plugin 目录下，重启 Vim 即可。

如果需要配置，请修改 cy.vim 文件前边的参数，其中有说明。

在Gvim中，默认使用 Alt-x 切换输入法， 在终端中，可以自由设置为 Ctrl-\ 等类似方式。
