扩展使用¶

穿越输入法当中用到的码表文件是cy.cy，直接可以用Vim编辑，其中你可以添加或修改自己的词汇。也可以转换成其它输入法的码表文件，然后就可以在Linux或Windows系统中像其它输入法一样使用了，当然也可以在Android手机中配合 VimTouch 使用。

颜色风格

以下是颜色风格文件，下载后放到 Vim 的 colors 目录下。当切换到中文状态时，自动变换光标颜色。

  • cyim ：暗色调风格
  • cyim-light ：明亮色调风格
